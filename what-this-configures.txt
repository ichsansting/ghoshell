WHAT GETS CONFIGURED
====================
A complete developer home environment (Linux & macOS), set up in one
apply. WORK and CLAUDE OAUTH sections are optional and off by default.

FORMAT — each entry:
  [action] name — description
      key = real value          (exact values, for replication)
ACTIONS:
  [install]        the tool is installed, default settings
  [config]         settings are written (tool itself not installed here)
  [install+config] installed and configured
  [deploy]         a file is placed into $HOME as-is
  [deploy:secret]  same, but stored encrypted; decrypted on apply

SHELL & TERMINAL
----------------
[install+config] fish — the shell
    plugin = done  (desktop notification when a long command finishes)

[config] fish abbreviations — typed word expands to the real command:
    ls  = "eza --icons"          ll  = "eza --icons -la"
    lt  = "eza --icons --tree"   cat = "bat"
    grep = "rg"                  find = "fd"
    top = "btop"                 htop = "btop"
    du  = "dust"                 df  = "duf"
    diff = "delta"               fm  = "yazi"
    claude = "BUN_JSC_useJIT=0 claude"   (JIT crash workaround)
    gst = "git status"
    glg = "git log --oneline --graph --decorate --all"
    gco = "git checkout"         gbr = "git branch"
    gunstage = "git reset HEAD --"
    glast = "git log -1 HEAD"

[install+config] starship — shell prompt
    palette = Catppuccin Mocha (standard published palette)
    segments, in order = username, hostname, directory, git branch,
      git status, nix shell, rust, python, node, command duration,
      line break, prompt character
    directory = bold lavender, show last 4 dirs, never truncate to repo
    cmd_duration = shown when a command took >= 2000 ms

[install+config] atuin — shell history
    auto_sync = true
    sync_frequency = "5m"
    search_mode = "fuzzy"
    flags = --disable-up-arrow   (up-arrow stays normal history)

[install+config] zoxide — smarter cd (jump to frequent directories)
    fish integration = on

[install+config] zellij — terminal multiplexer
    default_shell = "fish"
    mouse_mode = true
    theme = "catppuccin-mocha"
    keybinds = unbind "Alt f" and "Alt Right" in all non-locked modes
      (fixes Alt+Right word-jump across terminals)

EDITOR
------
[install+config] helix — the editor
    default editor = yes ($EDITOR/$VISUAL = hx)
    theme = "catppuccin_mocha"
    line-number = relative;  auto-pairs = on
    file-picker shows hidden files;  indent guides rendered
    cursor shape = bar (insert), block (normal), underline (select)
    statusline = mode/spinner/file-name/modified | version-control |
      diagnostics/selections/position/encoding
    languages wired to servers/formatters:
      nix        -> nixd, format with nixfmt, auto-format on
      bash       -> bash-language-server
      fish       -> fish-lsp
      markdown   -> markdown-oxide
      python     -> ty + ruff, format with "ruff format -", auto-format on
      typescript/javascript -> typescript-language-server
      rust       -> rust-analyzer (found on PATH)

[install] language servers/formatters — nixd, bash-language-server,
    fish-lsp, markdown-oxide, ty, ruff, typescript-language-server,
    nixfmt
    note: rust-analyzer intentionally NOT installed here — it comes with
    the rust toolchain (mise) so their versions always match
